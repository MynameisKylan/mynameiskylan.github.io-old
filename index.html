Homepage: The <h1> tag is used for page titles

The <p> tag is used for paragraph elements

<h2> and beyond are used for subheadings, sub-subheadings, etc.

<ul> and <ol> tags are used to make unordered and ordered lists, respectively

    Unordered:
  • This is
  • an unordered
  • list
    Ordered:
  1. This is
  2. an ordered
  3. list

We can express emphasis or importance. These tags have semantic value.
