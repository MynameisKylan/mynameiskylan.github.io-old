The <p> tag is used for paragraph elements

<h2> and beyond are used for subheadings, sub-subheadings, etc.

<ul> and <ol> tags are used to make unordered and ordered lists, respectively

Ordered:

  2. an ordered

We can express emphasis or importance. These tags have semantic value.

The HTML <blockquote> Element (or HTML Block Quotation Element) indicates that the enclosed text is an extended quotation.

The quote element — <q> — is intended for short quotations that don't require paragraph breaks.

Use the <abbr> tag for acronyms or abbreviations like HTML.

Marking up code:

  • <pre> for retaining whitespace/indentation
  • <code> for pieces of code
  • <var> for variable names
  • <kbd> for (keyboard) input into the computer
  • <samp> for computer output
